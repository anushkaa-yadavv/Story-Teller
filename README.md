## Advanced Story Teller

An AI-powered storytelling application that generates and narates creative and engaging stories based on user input.
This project demonstrates the use of Python and basic AI logic to simulate dynamic story generation.

---

## Features

* Generates unique and creative stories dynamically
* Improved storytelling logic for better flow and coherence
* User-friendly and easy to interact with
* Easily extendable with advanced AI/ML models

---

## Tech Stack

* Python
* NLP
* API integration
* Tkinter
* pyttsx3

---

##  How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/anushkaa-yadavv/Story-Teller.git
   ```

2. Navigate to the project folder:

   ```bash
   cd Story-Teller
   ```

3. Run the program:

   ```bash
   advance story teller.py
   ```
